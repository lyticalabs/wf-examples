# Lessons Learned: Observability

This document summarizes the key takeaways from the `observability` documentation.

- **`index.mdx`**: The Workflow DevKit provides a powerful set of observability tools, including a CLI and a web UI, for inspecting and debugging workflows. These tools allow you to view the status of workflow runs, inspect the inputs and outputs of steps, and trace the execution of your workflows. The observability tools can connect to any "World," making it possible to monitor workflows in both local development and production environments.
