# Lessons Learned: Getting Started

This document summarizes the key takeaways from the `getting-started` documentation.

- **`index.mdx`**: The `getting-started` documentation provides a clear and concise entry point for developers using Next.js. It directs users to the Next.js-specific guide, ensuring they have a smooth onboarding experience.

- **`next.mdx`**: The Next.js getting started guide provides a step-by-step tutorial for setting up and running your first workflow. It covers the essential configuration steps, such as wrapping your `next.config.ts` with `withWorkflow()`, and introduces the core concepts of workflows and steps. This hands-on guide is a great way to get up and running with the Workflow DevKit in a Next.js environment.
